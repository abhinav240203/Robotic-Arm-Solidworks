# 3-DOF Robotic Arm with Mechanical Gripper

A 3-DOF robotic arm designed and assembled in **SOLIDWORKS** with a mechanical parallel gripper. The project demonstrates CAD modeling, assembly design, and revolute joint mechanisms for robotic manipulation.

## Features
- 3-DOF robotic arm
- Mechanical parallel gripper
- Fully assembled in SOLIDWORKS
- Modular link design
- Ready for motion simulation

## Software
- SOLIDWORKS

## Applications
- Pick-and-place robotics
- Robot kinematics
- CAD and assembly practice
